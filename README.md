## Карта
 В файл Input.txt вы можете нарисовать свою карту.
 Введите размер поля N * M, а затем саму карту:
 
 '#' - стены,
 
 '.' - пустое пространство,
 
 '@' - ваш персонаж.
 
 уже готовая карта находиться в файле [Input.txt](https://github.com/ARGimadeyev/pseudo-3d/edit/main/Input.txt)
